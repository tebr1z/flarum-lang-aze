# az Aze Experience Pack for Flarum
[![Latest Stable Version](https://img.shields.io/packagist/v/flarum-lang/aze?color=success&label=stable)](https://packagist.org/packages/flarum-lang/aze)  [![License](https://img.shields.io/packagist/l/flarum-lang/aze)](https://packagist.org/packages/flarum-lang/aze) [![Total Downloads](https://img.shields.io/packagist/dt/flarum-lang/aze)](https://packagist.org/packages/flarum-lang/aze/stats) [![Monthly Downloads](https://img.shields.io/packagist/dm/flarum-lang/aze)](https://packagist.org/packages/flarum-lang/aze/stats) 

**Always up-to-date, always improving Flarum language experience!**

We are now delivering free language updates through the this package. This means we can continuously improve your local language and automatically send these updates to your site. By installing the aze Experience Pack extension, the Flarum text in your language will always be up-to-date.

Interested in helping make Flarum text in your local language better? You can easily provide suggestions on text improvements using the GitHub. Just visit "[Issues](https://github.com/flarum-lang/aze/issues)" page on our extension page.

___



## Installation

```bash
composer require flarum-lang/aze
```
